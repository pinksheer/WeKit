# 红包与支付

红包与支付功能提供微信支付和红包的增强体验。

## 功能列表

- [自动抢红包](auto-open-red-packets.md) — 自动拆红包并 Toast 提示金额和来源, 支持自动回复
- [自动接收转账](auto-accept-transfers.md) — 自动确认收款, 支持白名单/黑名单过滤
- [指纹支付](fingerprint-pay.md) — 使用指纹替代支付密码完成支付确认, 基于 TEE 加密存储
- [允许领取私聊红包](allow-private-chat-receive-outgoing-red-packets.md) — 允许打开私聊中自己发出的红包
- [修改显示余额](modify-wallet-balance-display.md) — 伪装钱包首页显示的余额文字
- [修改转账显示余额](modify-transfer-wallet-balance-display.md) — 伪装转账时显示的余额文字
