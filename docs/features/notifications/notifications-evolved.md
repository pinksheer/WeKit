# 通知进化

> 让微信的新消息通知更易用。1. 「快速回复」按钮 2. 「标记为已读」按钮 3. 使用原生对话样式 (MessagingStyle)

## 类别

通知

## 类型

可开关

## 描述

增强微信的通知系统, 提供以下三项改进:

1. **快速回复按钮** — 在通知栏中直接回复消息, 无需打开微信
2. **标记为已读按钮** — 在通知栏中一键将消息标记为已读
3. **原生对话样式 (MessagingStyle)** — 使用 Android 系统原生的 MessagingStyle 渲染通知, 展示更清晰的对话结构

## 使用方法

在模块设置中启用
