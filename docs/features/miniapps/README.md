# 小程序

小程序功能提供广告移除、限制解除等增强体验。

## 功能列表

- [移除开屏广告](remove-splash-ads.md) — 跳过小程序启动时的广告页面
- [移除嵌入广告](remove-embedded-ads.md) — 移除小程序界面中的嵌入广告
- [移除视频广告](remove-video-ads.md) — 跳过小程序中的视频广告
- [去除菜单限制](remove-menu-limits.md) — 解锁小程序右上角菜单的限制选项
- [伪装宿主版本](spoof-host-version.md) — 解决因微信版本较低导致部分小程序无法使用的问题
- [跳过启动页面](skip-splash.md) — 跳过小程序启动加载页, 变相去广告 [实验性]
